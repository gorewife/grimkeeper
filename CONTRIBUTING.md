# Contributing

## Report a Bug

[Open an issue](https://github.com/gorewife/grimkeeper/issues) and describe what went wrong. Screenshots help.

## Request a Feature

[Open an issue](https://github.com/gorewife/grimkeeper/issues) and describe what you want added.

## Submit Code

Fork the repo, make your changes, and open a pull request. Test your changes first.

## Questions

Contact `hystericca` on Discord.
