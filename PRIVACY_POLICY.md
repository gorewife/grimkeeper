# Privacy Policy

**Last Updated:** November 18, 2025

## Overview

This Privacy Policy explains what data Grimkeeper collects, how it's used, and how it's stored.

## Data We Collect

### Server Configuration Data
- **BOTC Category ID** - To identify voice channels for game management
- **Town Square Channel ID** - Destination for player call commands
- **Announcement Channel ID** - Where bot updates are posted

### User Data
- **User IDs** - Discord user IDs for tracking follows, DND status, and timers
- **Nicknames** - Temporary prefix modifications for role identification
- **Voice Channel States** - Current voice channel for shadow following feature

### Game Session Data
- **Grimoire Links** - URLs to online grimoires (per server)
- **Shadow Follow Relationships** - Who is following whom
- **Do Not Disturb Status** - Users who opt out of being followed
- **Active Timers** - Scheduled call timers and their creators
- **Player Snapshots** - Previous player list states for join/leave tracking

## How We Use Your Data

- **Server Configuration** - To enable core bot functionality
- **User IDs** - To manage shadow following, timers, and DND preferences
- **Voice States** - To move spectators when shadow following is active
- **Grimoire Links** - To share game resources with server members

## Data Storage

- All data is stored locally in JSON files on the bot's host server
- Data is not shared with third parties
- Data is not sold or used for advertising
- No personal information (names, emails, etc.) is collected beyond Discord IDs

## Data Retention

- Server configuration data persists until manually removed with bot commands
- User preference data (DND, follows) persists until changed by the user
- Grimoire links persist until updated or the bot is removed from the server
- Timer data is deleted when timers expire or are cancelled

## Data Deletion

You can delete your data at any time:
- **Shadow follows:** Use `*unspec` command
- **DND status:** Use `*dnd` command to toggle off
- **Server data:** Remove the bot from your server (deletes all server-specific data)
- **Manual deletion requests:** Contact hystericca on Discord

## Third-Party Services

Grimkeeper does not use any third-party analytics, tracking, or data collection services.

## Discord API

Grimkeeper uses the Discord API and is subject to Discord's Privacy Policy. We only access data necessary for bot functionality as permitted by Discord.

## Children's Privacy

Grimkeeper does not knowingly collect data from users under 13. Discord's Terms of Service require users to be 13 or older.

## Data Security

- Data is stored on a secure server
- Access is restricted to the bot developer
- No encryption is used for stored data (Discord IDs are not sensitive)

## Your Rights

You have the right to:
- Request what data we have about you
- Request deletion of your data
- Opt out of certain features (shadow following, DND)

## Changes to This Policy

We may update this Privacy Policy at any time. Changes will be reflected with a new "Last Updated" date.

## Contact

For privacy concerns or data deletion requests, contact: hystericca on Discord
